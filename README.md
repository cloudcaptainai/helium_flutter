# helium_flutter

This repo contains two packages for Helium Flutter integration:
1. **./packages/helium_flutter** the main SDK package
2. **./packages/helium_revenuecat** an alternate package for easy integration with RevenueCat.

## How to Integrate

Follow our [official documentation here](https://docs.tryhelium.com/sdk/quickstart-flutter) to integrate Helium into your Flutter app!
